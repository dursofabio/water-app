# Mockup US-005 - Lista ultimi pagamenti

Mockup isolato per la dashboard pubblica di AcquaApp.

- `index.html`: schermata principale con saldi, ultimi carichi e ultimi pagamenti.
- `shared.css`: token visivi e primitive condivise, coerenti con US-003 e US-004.
- `dashboard-payments.css`: layout e componenti specifici della lista pagamenti.
- `app.js`: piccole interazioni statiche per espandere i carichi, simulare un nuovo pagamento e vedere lo stato vuoto.
